# Feature List

## Unique Features

- Dual-Mode Identity (Pro/Social): Lets users switch between professional and social personas in one account with separate vibe and content context.
- BLE Nearby Discovery: Finds nearby users over Bluetooth without requiring internet connectivity for initial detection.
- Offline Visibility Broadcasting: Advertises your presence over BLE so nearby users can discover you in real time.
- BLE + GPS Hybrid Discovery: Supports both local Bluetooth proximity and internet-based GPS radius discovery in one screen.
- Mode-Aware Content Streams: Filters feed, chats, and interactions by active mode to keep Pro and Social experiences distinct.
- Mood Status Bubble in Chat: Shows short-lived mood indicators with expiry time directly in messaging surfaces.
- Selected-Connections Post Visibility: Allows a post to be visible only to a custom-picked subset of your connections.
- Mesh Chat Entry Point: Provides an offline-first messaging pathway for local communication scenarios.
- Connection Queue for Offline Retry: Queues connection requests when offline and sends them automatically when connectivity returns.
- Nearby Context Actions: Lets users quickly message, view profile, or connect directly from nearby discovery results.

## Not Fully Unique Features

- Direct Messaging: Enables one-to-one real-time chat between users.
- Group Chat: Allows multiple users to communicate in shared conversation rooms.
- Message Reactions: Lets users respond to messages with emoji reactions.
- Reply Threads: Supports replying to specific messages with contextual previews.
- Date Separators and Timestamps: Displays message day breaks and human-readable send times.
- Typing Indicators: Shows when other users are actively typing in chat.
- Read Receipts: Indicates whether sent messages have been seen by recipients.
- Message Edit and Delete: Allows users to edit recent messages or soft-delete them.
- Pinned Group Messages: Highlights important group messages by pinning them at the top.
- Group Roles (Creator/Admin/Member): Adds role-based privileges for managing group behavior.
- In-Chat Search: Enables keyword filtering of chat history in direct and group chats.
- Stories, Reels, and Feed Posts: Supports short-form and feed-style social content publishing.
- Follow/Connection Model: Manages relationship state like pending, accepted, and connected.
- Profile Customization: Lets users maintain bios, avatars, and personal details.
- Push Notifications: Sends event updates such as activity and message alerts to devices.
- **Music Library Integration**: Browse and select from a library of songs (demo + user-uploaded) to add to posts and stories.
- **Audio Playback**: Play/pause music with progress bar and seek controls in posts and stories.
- **Song Upload**: Upload custom audio files (MP3, M4A, AAC, WAV, OGG) to personal music library using Cloudinary.
- **Music Search**: Filter songs by name or artist in music selector widget.
