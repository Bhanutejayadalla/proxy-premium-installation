How to Install (Android)

  1. Tap Download APK above
  2. Open your Downloads folder
  3. Tap proxi-premium.apk
  4. Allow Install from Unknown Sources if prompted
  5. Tap Install — done! 🎉

🔵 Mesh Chat — New in v3.1

  1. Open any chat or the Messages screen
  2. Tap the Bluetooth icon in the top bar
  3. Toggle Mesh ON — BLE scanning starts
  4. Type and send messages — they travel over BLE with no internet!
  5. Messages auto-sync to Firebase when back online ☁
